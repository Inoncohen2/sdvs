# Lumina Minimalist Blog

A professional, minimalist blog architecture designed with a focus on typography, whitespace, and user experience.

## Features
- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **PWA Ready**: Offline support via Service Workers.
- **SEO Optimized**: Pre-configured meta tags for search engines and social sharing.
- **Modern Stack**: Clean HTML5, CSS3, and Vanilla JS.

## Setup
Simply open `index.html` in your browser or deploy to Vercel/Netlify.
